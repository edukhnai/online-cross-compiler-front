Online Cross Compiler with Parallel Programming Support

This is the Online Cross Compilation system with Parallel Programming support.
Authors: Ekaterina Dukhnai, Olga Kostyukova, Lilia Kudelina.
Supported PLs: C, C++ OpenCL 1.2, OpenCL 1.2 C++ API, OpenCL Kernel (.cl) compilation into PTX (.ptx) CUDA file for NVIDIA GeForce GTX 1050ti and Tesla K80 GPGPUs, OpenCL Kernel (.cl) compilation into binary (.bin) file for AMD Radeon RX5700 GPU.
